# 6주차 — 우선순위 큐 다익스트라

양의 가중치 단방향 그래프에서 **최소 힙**을 쓰는 다익스트라를 다룹니다. 5주차의 O(V²) 구현과 달리, **V·E가 큰 입력**에 맞는 **O((V+E) log V)** 유형에 초점을 둡니다.

## 포함 파일

| 파일 | 설명 |
|------|------|
| `main.ipynb` | 문제 설명, TODO·정답 코드, Mermaid 정리 |
| `main.py` | 문제 2 유형의 간결 예시(최단 거리 + `parent`로 경로 복원) |

## 문제 요약

1. **디랩이의 모든 도시까지 이동 계획** — 시작점 `K`에서 **모든 정점**까지 최단 거리만 출력. 참고: [BOJ 1753 최단경로](https://www.acmicpc.net/problem/1753).
2. **최소 이동 비용과 경로** — 시작~도착 **한 쌍**의 최단 비용, 간선 수, **정점 경로**까지 출력. 다익스트라 중 `parent` 배열로 역추적. 참고: [BOJ 11779 최소비용 구하기 2](https://www.acmicpc.net/problem/11779).

## 노트북 구성

`main.ipynb`는 문제마다 아래 흐름을 따릅니다.

1. 문제 설명
2. TODO 코드(학습자용)
3. TODO 주석 포함 정답 코드
4. 간결 정답 코드(제출용)
5. Mermaid 다이어그램(아이디어·흐름·경로 복원 등)

## 실행 방법

- 노트북: `jupyter notebook week6/main.ipynb`
- 스크립트(표준 입력 필요): `python3 week6/main.py`
