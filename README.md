# algorithm_lv4

11주차 알고리즘 학습 내용을 주차별로 정리하는 저장소입니다.  
모든 주차를 길게 설명하기보다, 각 주차의 핵심 주제와 학습 산출물을 간단히 기록하는 것을 목표로 합니다.

## 현재 구조

```text
algorithm_lv4/
├─ week1/
│  └─ main.ipynb
├─ week2/
│  └─ main.ipynb
└─ README.md
```

## 주차별 개요 (요약형)

- `week1`: 우선순위 큐 기초 연습 (절댓값 기준 최소 힙 처리)
- `week2`: 우선순위 큐 응용 2문제 정리 (`숫자 카드 정렬 게임`, `쇼핑몰`)
- `week3` ~ `week11`: 주차별 핵심 문제/주제 중심으로 순차 추가

## 노트북 구성

`**/main.ipynb`는 문제마다 아래 학습 흐름으로 구성됩니다.

1. 문제 설명
2. TODO 코드(학습자용)
3. TODO 주석 포함 정답 코드
4. 간결 정답 코드(제출용)
5. Mermaid 다이어그램(핵심 아이디어/흐름도/예제 추적)

## 실행 방법

- 파이썬 스크립트 실행: `python3 <파일명>.py`
- 주피터 노트북 실행: `jupyter notebook week2/main.ipynb`
